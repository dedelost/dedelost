Python——scrapy之抓去丁香园药品详情信息

发表于 2016-10-24   |  

安装前准备

主页献上scrapy
安装要求：
Python 2.7 or Python 3.3+
Works on Linux, Windows, Mac OSX, BSD
也就是说Mac OSX开发，Linux部署是可以的了。语言版本拥抱3.x版本。

安装

$ pip install scrapy

踩坑：由于开发环境使用的是PyCharm,安装的话。搜索安装添加 ok。详细不知道的同学我会专门写一个教程，原因是我百度没找见，原谅我不用google，原因是虽然写过google的很多web产品，然而还是没找到特别突出的点。PyCharm第三方包管理和部署还有很多人写Python不喜欢用IDE。

言归正传：想用3.X系列，但是mac 默认2.x系列，软连接啥的没发做，root权限不够。。。哈哈哈很奇怪吧！算了委屈一下，现在2.x上折腾下，成功了再逼逼其他方式。

这时候开发工具变成了 SubLime + 终端了。哈哈哈。。

环境好了，mac安装的时候有时候会爆出一个权限问题的错误，这个时候执行这个

pip install --ignore-installed six

可以解决这个问题。

开动

安装完成之后先走一个demo。

scrapy startproject scrapy_DXY
scrapy crawl dmoz

关键两部，也就拉下来了URL对应的两个页面，并写到本地，接下来选择器过滤。比较繁琐了。。。来个官方教程

scrapy工具

scrapy startproject myproject // 创建项目
scrapy genspider mydomain mydomain.com // 创建spider
