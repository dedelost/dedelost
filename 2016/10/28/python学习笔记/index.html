内建函数详解

bisect

操作要求列表是已经排序过后的列表

import bisect
print dir(bisect)

[‘builtins‘, ‘doc‘, ‘file‘, ‘name‘, ‘package‘, ‘bisect’, ‘bisect_left’, ‘bisect_right’, ‘insort’, ‘insort_left’, ‘insort_right’]

  • bisect 

    bisect.bisect(data,7)
查找列表数值位置，并返回位置。不存在超过最大值则返回，len（列表），小于最小值则返回0

  • insort 

    bisect.insort(data,7)
为排序后的列表插入元素，按照排序自动插入
  • insort_left
  • insort_right
  • bisect_left

  • bisect_right 

    对应left以及right 方法，只是优先插入,和优先返回坐标值！

map

可以叫爸爸的函数、

  • 批量计算：

    def add(x):
    … return x+1
    …
    li = [1,2,3]
    map(add,li)
    [2, 3, 4]

  • 批量并行计算:

    >>> def add(a,b):
...     return a*100+b*10
...
>>> li1=[1,2]
>>> li2=[3,4]
>>> map(add,li1,li2)
[130, 240]

  • 第一参数为None:

    map(None,li1,li2)
    [(1, 3), (2, 4)]

文件和流

open(r'name','wb',0/1) 
第三个参数：0 无缓冲，直接操作disk。 
1 有缓冲，使用flush或者close方法才将内存中缓冲的信息写入disk
